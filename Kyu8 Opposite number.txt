/* description 

Opposite number

Very simple, given an integer or a floating-point number, find its opposite.

Examples:

1: -1
14: -14
-34: 34

/

JavaScript solution1 : 

function opposite(number) {
   return -number 
}

JavaScript solution2 : 

function opposite(number) {
   return number * -1 
}

JavaScript solution3 : 

const opposite = number => -number
